# 🚀 Multi-Threaded Pipeline (C)

A multi-threaded pipeline system in C that processes data across concurrent stages using POSIX threads.

---

## 🧠 Key Features

- Multithreading with `pthread`
- Producer–consumer pattern
- Mutexes & condition variables for synchronization
- Thread-safe shared buffers

---

## 🛠️ Build & Run

    make
    ./pipeline <testcase>

Example:

    ./pipeline Testing/Test1

---

## 📁 Structure

- pipeline.c
- pipeline_driver.c
- Makefile
- Testing/
